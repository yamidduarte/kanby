Kanby is an open-source Progressive Web Application (PWA) designed around reliability and speed of NoSQL databases.

Inspired by such project management suites as Trello, Notion, and Monday, Kanby takes aim at the exponentially increasing complexity of premium solutions offered today. Kanby's mission is to provide a free service that's laser-focused on speed, responsiveness, and scalability of its Kanban board offering. Deployed as a containerized serverless microservice on Google Cloud Run, the purpose of this project is to prototype, build, and integrate a highly scalable architecture while leveraging the fastest path to production.

Kanby is built and maintained in TypeScript by Yamid Duarte, a one-person team using Angular, Firebase, Node.js, and Docker. For more information about Kanby, please visit the repository mirrored on GitHub and BitBucket.
